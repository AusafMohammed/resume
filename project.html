VchatApp

GITHUB LINK

I spearheaded the development of an Android chat application using Java and Firebase, offering users a seamless communication platform. The app features two main fragments: one for individual chats and the other for group interactions. With Firebase Authentication, users can securely sign up and log in. In the individual chat fragment, users can view all registered users, initiate real-time chats, engage in video/voice calls, and share live locations. The group chat fragment enables users to create groups, add members, and exchange messages in real-time. The menu offers options for profile management, including profile picture upload and bio editing. This project demonstrates my proficiency in Android development, Firebase integration, and user-centric design, providing a feature-rich chatting experience for users.

MyInstaApp

GITHUB LINK

This Android project offers a straightforward sign-in/log-in service and comprises three fragments within the second activity: Profile, Users, and Posting Images. In the Profile fragment, users can view their username, bio, and posts stored on the Back4App server. The interface is designed with a smooth scrolling experience using LinearLayouts. The Users fragment displays a ListView of all registered users, enabling users to click and view posts made by others in the third activity. The Posting Images fragment allows users to upload images with captions. A logout feature is accessible from the toolbar and also triggered by the back button.

"MyTwitterApp" is an Android application developed using Back4App for backend services and cloud integration. The app features a seamless sign-in/log-in interface, with user credentials securely stored on the Back4App server. It offers a user-friendly platform for browsing users, posting tweets, and viewing tweets from followed users, utilizing RecyclerView and CardView for organized tweet presentation. MyJokesApp

GITHUB LINK

"MyJokesApp" is an Android application designed to provide users with a unique joke experience sourced from a joke API with each button click. The app's user interface features a card view displaying jokes alongside share and like buttons represented as vector assets. Clicking the like button marks the joke as a favorite, storing it in a separate activity named FavJokesActivity accessible through the toolbar. Liked jokes are saved in an SQLite database for easy retrieval. Additionally, users can share jokes via WhatsApp, Facebook, or other messaging platforms using the share button. The toolbar also offers an option to view favorite jokes, presenting them in a RecyclerView. Users can share and delete jokes directly from this activity by sliding the joke to the right. This project showcases proficiency in Android development, including UI design, API integration, database management, and sharing functionalities.
